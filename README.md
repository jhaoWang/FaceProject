# 人脸修复 Demo 软件项目说明

本项目是一款基于 LaFin 算法的遮挡人脸修复桌面端演示 Demo，为算法工程化落地提供了完整的验证与展示方案。

<img width="1710" height="1109" alt="image" src="https://github.com/user-attachments/assets/fb86b691-5f39-4ddf-bb44-197c35b75913" />


系统采用前后端解耦的设计模式，实现了交互层与推理层的职责分离：

- **交互层**：基于 Qt 构建轻量化桌面界面，支持原始图像、人脸关键点、遮挡掩码三类输入的快速配置，以及修复结果的实时预览与对比，降低了算法验证的操作门槛；
- **推理层**：核心采用 LaFin 人脸修复算法，模型通过 Python 训练完成后导出为 ONNX 格式，再基于 C++ 与 ONNX Runtime 构建高性能推理引擎，封装为独立 DLL 供前端调用，既保留了深度学习模型训练迭代的便利性，又实现了推理阶段的高性能部署，有效支撑了遮挡人脸图像的快速修复推理。
